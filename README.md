## Project setup

### Compiles and hot-reloads for development
```
yarn start
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### Start json-server
```
yarn dbwatch
```

### Compiles and hot-reloads for development and start json-server
```
yarn serve
```
